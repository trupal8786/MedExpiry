# 💊 MedExpiry

**Hacktivate Syndicate | Smart India Hackathon 2025**

🌐 **Live Demo:** [https://medexpiry.onrender.com](https://medexpiry.onrender.com)

MedExpiry is a hardware-free web application that empowers households to actively track the expiry dates of their medicines and provides a direct pipeline to donate unused, valid medications to verified NGOs.
